# CleanTemp

CleanTemp is a fast and simple tool for cleaning temporary files on Windows  
It has a clean (cli) interface JUST to free up space and remove unnecessary (temp) files that wont damage your PC

## Features

- Cleans user temp folder with (`cleantemp`)  
- Cleans windows temp folder with (`cleantemp`)  
- Cleans prefetch folder with (`cleanprefetch`)  
- Cleans Windows temp, User temp and prefetch with (`fullclean`)  
- Empties Recycle Bin (subject to Windows permissions)   
- Its a CLI with colored warnings and cyan, clean looking prompt
- `credits` for credits, itz me :D 

## How2Use

1. Run `cleantemp.exe` or the source code  
2. Use these `cleantemp`, `fullclean`, `cleanprefetch`, `credits`, or `exit`  
3. You can exit with `exit` or clean folders again
4. If there is an issue or you dont trust the program just open an Issue in "Issues" orrrrrrr read the source code, its very simple and basic
